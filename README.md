# amazon_3statement-model
This repository contains my first attempt at building a 3-statement financial model using Amazon’s financial data (2021–2025). It includes all 3 the statements with basic linkages between them. The model is a work in progress and will be improved with better assumptions, error checks, and more dynamic structuring over time.
This repository contains my first attempt at building a 3-statement financial model using Amazon’s financial data (2021–2025).
The model includes:
- Income Statement
- Balance Sheet
- Cash Flow Statement
- Basic linkages between statements
This is a work in progress and will be continuously improved with better assumptions, validations, and additional features.
